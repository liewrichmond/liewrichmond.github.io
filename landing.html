Java Spreadsheets

A Java Swing Application

Motivation

This Java Spreadsheet program was written for the Object Oriented Design class of Spring 2019.

MVC Architecture

Written with an MVC architecture, the code aims to decouple code as much as possible, such as isolating Swing components to the View only

An Object Oriented Exercise

The program is an exercise in Object Oriented Design principles and leverages the use of Object Oriented design patterns to construct an extensible, flexible and maintainable codebase.

Features

  • Infinite Scrolling
  • Cell References
  • Rectangular References
  • Cycle Detection
  • Auto Update
